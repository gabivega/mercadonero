# Nero Marketplace — Frontend (MVP)

Proyecto frontend minimal para el MVP del marketplace multivendor.

Quick start

1. Instalar dependencias:

```bash
cd "c:\Users\gabil\OneDrive\Desktop\Diseño Web\nero\MVP\frontend"
npm install
```

2. Ejecutar en modo desarrollo:

```bash
npm run dev
```

Notas
- Proyecto generado con Vite + React (JavaScript)
- Tailwind CSS configurado con `darkMode: 'class'`
- Redux Toolkit incluido con un `theme` slice para togglear tema
